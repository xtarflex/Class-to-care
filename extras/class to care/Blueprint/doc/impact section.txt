Icon Animations
--------------

1. Health Workers Icon (Users)
- Animation: "The Gentle Pop-in"
- Implementation:
  * Initial state: opacity: 0, scale(0.5)
  * Final state: opacity: 1, scale(1)
  * Timing: ease-out
  * Class to add: .users-icon

2. Health Centres Icon (FirstAidKit)
- Animation: "The Supportive Pulse"
- Implementation:
  * Main kit fades in first
  * Cross element pulses (scale 110% -> 100%)
  * Required class: .cross
  * Timing: 0.3s delay on cross animation
  * Cross element needs: class="cross"

3. Students Icon (GraduationCap)
- Animation: "The Tassel Swing"
- Implementation:
  * Cap body fades in
  * Tassel swings (rotate -5deg -> +5deg)
  * Required class: .tassel
  * Transform origin: top center
  * Tassel element needs: class="tassel"

4. Operation Time Icon (CalendarCheck)
- Animation: "The Check Mark Draw"
- Implementation:
  * Calendar grid fades in
  * Check mark draws itself
  * Required class: .check-mark
  * Uses stroke-dasharray/offset
  * Check mark element needs: class="check-mark"

Technical Implementation Notes:
-----------------------------
- All animations trigger on scroll into view
- Use Intersection Observer API
- SVG files require manual class additions
- Animations should complete within 1.2s
- Add subtle ease-out to all transitions
- Ensure fallback static display for reduced motion preferences
